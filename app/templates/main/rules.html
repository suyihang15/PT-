站点规则

一、总则
  • 1.1 本站是私有BT种子分享站点，请遵守所有规则。
  • 1.2 禁止分享违法内容，包括但不限于色情、暴力、政治敏感内容。
  • 1.3 尊重他人，禁止辱骂、人身攻击等不当行为。
二、分享率规则
  • 2.1 用户应当维持良好的分享率（上传量/下载量）。
  • 2.2 当分享率低于 0.4 时，将无法下载新种子。
  • 2.3 PowerUser 及以上等级可享受更多权限。
  • 2.4 免费种子 (Freeleech) 不计算下载量，但仍计算上传量。
三、Hit & Run (H&R) 规则
  • 3.1 下载完成后，需在 7天 内累积做种 72小时 或分享率达到 1.0。
  • 3.2 未满足H&R规则的种子将被标记为违规。
  • 3.3 累计3次H&R违规可能导致账号被封禁。
四、种子发布规则
  • 4.1 上传种子前请确认没有重复。
  • 4.2 种子描述应包含完整的资源信息。
  • 4.3 禁止上传含有病毒或恶意软件的种子。
